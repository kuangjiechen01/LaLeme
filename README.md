# 拉了么

一个移动端优先的每日排便监督 Web App / PWA，用于私密记录排便情况、观察频率和节律，并在频率偏低时给出温和提醒。

## 技术栈

- React + Vite
- Tailwind CSS v4
- Framer Motion
- Recharts
- Vite PWA
- localStorage

## 启动

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 功能概览

- 首页 Dashboard
- 快速打卡底部抽屉
- 历史记录筛选、编辑、删除
- 统计分析与节律观察
- 提醒阈值、文案风格、主题模式设置
- JSON 导入 / 导出
- PWA 安装支持

## 数据说明

所有数据默认保存在浏览器本地。项目自带示例数据，首次打开即可看到完整产品效果。
